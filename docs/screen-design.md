# 日報一覧画面

URL:
/reports

## 要素

・ヘッダー
・プロフィールリンク
・日報作成ボタン
・検索フォーム
・チーム参加ボタン（参加している場合はチーム名を表示）
・日報カード一覧
・ページネーション

## 日報カード表示項目

・タイトル
・投稿者（所属しているチーム名）
・日付
・カテゴリ
・本文冒頭
・コメント数
・作成日時

[日報作成ボタン]

[日報カード]

# 日報作成画面

URL:
/reports/new

## 要素

・タイトル入力
・日付選択
・カテゴリ選択
・本文入力
・公開範囲切り替え
・投稿ボタン
・キャンセルボタン

## バリデーション

タイトル：
必須・50文字以内

本文：
必須・2000文字以内

---

# 日報詳細画面

URL:
/reports/[id]

## 要素

・タイトル
・投稿者名
・アバター
・日付
・カテゴリ
・本文
・コメント一覧
・コメント投稿フォーム

自分の日報のみ：

・編集ボタン
・削除ボタン

# 日報編集画面

URL:
/reports/[id]/edit

・更新ボタン
・削除ボタン

# レスポンシブ対応

スマホ：

・1カラム表示
・カード縦並び

PC：

・最大幅制限（max-width）
・中央配置

対応幅：

スマホ：
375px以上

タブレット：
768px以上

PC：
1024px以上

# daily report app Portfolio
Daily Report App を開発しました。チーム内で日々の業務内容や学習内容を共有し、コミュニケーションを活性化できる日報管理サービスです。日報は文章だけでなくカテゴリ分類や検索機能にも対応しており、必要な情報へ素早くアクセスできます。実務では情報共有ツールを使う機会が多いため、現場に近いテーマとして制作しました。技術スタックは Next.js（App Router）・TypeScript・Tailwind CSS・Supabase を採用し、認証・DB・ホスティングまで一貫して構築しました。工夫した点は、PC・スマホ両対応のレスポンシブ設計と、直感的に操作できるUI設計です。また、バリデーションやエラーハンドリングも実装し、安心して利用できる設計を意識しました。苦労した点は、認証状態管理やダークモード切替時の表示崩れ、検索条件と状態管理の整理です。開発を通じて、設計段階でのデータ構造の重要性、保守性を意識したコンポーネント分割、GitHub Actions を用いたCI/CDやPlaywrightによるE2Eテストなど、実務に近い開発フローを学ぶことができました。
